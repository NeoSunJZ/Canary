# 安装 Canary

# Python / Pytorch 版本

Canary至少需要Python 3.8，但建议使用Python 3.10或更高版本。

请事先安装PyTorch，Canary至少需要PyTorch 2.0.1，但建议使用PyTorch 2.1.1或更高版本。

# 安装Canary核心框架稳定版本

PyPI上提供了稳定版本：

pip install canary-sefi

对于使用Canary Lib的用户，应安装额外的库：

pip install canary-library

# 验证安装

以下命令可用于验证Canary SEFI是否已成功安装。如果一切正常，它应该将Canary的版本打印到命令行：

python -c "import canary; print(canary.__version__)"
1.0.0

以下命令可用于验证Canary Library是否已成功安装。如果一切正常，它应该将Canary Library的版本打印到命令行：

python -c "import canary_library; print(canary_library.__version__)"
1.0.0
