An Adversarial Robustness Evaluation Platform for Deep Models on Image Classification

最广泛的方法库

Canary为研究人员提供集成了29种被广泛使用的攻击方法、4类20种常用的防御策略与18种人工智能模型的内置库。

最自由的集成框架

Canary为研究人员提供快速集成框架SEFI，仅需添加几行注解即可将自行实现的攻击、防御方法或模型接入平台，完成测试评估。

最全面的测评

Canary为研究人员提供方法/模型无关的12个攻击方面、3个模型方面和5个防御方面的度量指标，并提供基准位次排序。

# 评估也可以开箱即用

仅需如下代码即可利用MI-FGSM对Alexnet进行测试

    # 初始化
    batch_manager.init_batch()

    # 配置
    config = {
      "dataset_size": 1,
      "dataset": "ILSVRC-2012", # 配置数据集
      "model_list": [ "Alexnet(ImageNet)" ], # 指定参与模型
      "attacker_list": {
        "MI-FGSM": ["Alexnet(ImageNet)"], # 指定攻击方法与对应攻击模型
      },
      "attacker_config": {
        # 指定攻击方法参数
        "MI-FGSM": {"alpha": 0.005, "epsilon": 8 / 255, "pixel_min": -3, "pixel_max": 3, "T": 1000, "attack_type": "UNTARGETED", "tlabel": None}
      }
    }

    # 执行测试
    security_evaluation = SecurityEvaluation(config)
    security_evaluation.attack_full_test()
